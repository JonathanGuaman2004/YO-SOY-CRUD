# YO-SOY-CRUD

Repositorio académico que integra dos aplicaciones:

1. **epn-event-manager**: backend en NestJS encargado de registrar eventos del sistema.
2. **crud**: aplicación CRUD del tema escogido desarrollada con Express, SQLite y frontend web.

El objetivo del proyecto es demostrar los 4 tipos de mantenimiento sobre una aplicación con fallas iniciales, agregando validaciones, persistencia, registro de eventos y mejoras de calidad.

## Orden recomendado para ejecutar el proyecto

Primero ejecutar el servicio de eventos:
```bash
cd epn-event-manager
npm run start:dev
```

Luego, en otra terminal, ejecutar el CRUD de misiones:

```bash
cd xxx-crud
npm start
```
